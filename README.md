# React Project Template

A simple react project template using [create-react-app](https://create-react-app.dev/).

## Packages used

- `create-react-app@5`
- `craco@7`
- `react@18`
- `react-router-dom@6`
- `antd@5`
- `@reduxjs/toolkit`, `react-redux`
- `sass`
- `husky`, `lint-staged`
- `eslint`, `prettier`
- `typescript`
